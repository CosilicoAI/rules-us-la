# rules-us-la

Louisiana rules in Akoma Ntoso XML format.

## Structure

- `statutes/` - Louisiana Revised Statutes in Akoma Ntoso XML
- `regulations/` - Louisiana Administrative Code in Akoma Ntoso XML

## About

This repository contains machine-readable versions of Louisiana legal texts encoded in [Akoma Ntoso](http://www.akomantoso.org/) XML format, enabling programmatic access for legal research, policy analysis, and rules-as-code applications.

Part of the [Cosilico](https://cosilico.ai) ecosystem.
